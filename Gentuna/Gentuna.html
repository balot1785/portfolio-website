Structured Cabling with PA-BX system

June. 2020-Jan. 2021

This project aimed to provide a reliable network infrastructure for General Tuna's new canning facility. We installed a structured cabling system and an analog PABX to ensure efficient operations, communication, and data management.

Project Deliverables

  • 200 Network Nodes: For the Local Area Network of the Offices.
  • 50 Boxes of Cat6 Cables: To support Gigabit Ethernet and PA-BX systems.
  • 5 48-Port Managed Switch: Facilitates data transmission between the devices connected throughout the network.
  • 26 Analog Handsets: For communication within the Offices.
  • PABX Controller: Manages the routing of calls from the handsets.

Project Scope

  • Structured Cabling System
  • Infrastructure Design: Design and layout of a comprehensive structured cabling system to support data transmission throughout the facility.
  • Cable Installation: Installation of high-quality cables, patch panels, and keystone jacks to ensure reliable connectivity.
  • Equipment Installation: Mounting of network cabinets and junction terminals for equipment housing.
  • Testing and Commissioning: Thorough testing of the structured cabling system to verify proper functionality and performance.
  • Analog PABX System
  • System Installation: Installation of the PABX system in a designated location within the facility.
  • Configuration: Configuration of the PABX system to support various communication features.
  • Testing and Commissioning: Thorough testing of the PABX system to verify proper functionality and performance.
