Locally Deployed Booking System for Grand Palmera Hotel

Aug. 2024-Oct. 2024

The staff at Grand Palmera Hotel require an efficient booking system to streamline their workflow and provide a clear, real-time overview of available slots for seamless handling of booking inquiries.

Project Deliverables

  • A locally Deployed booking system for the Grand Palmera Hotel.

Project Scope

  • Create a booking system the is user-friendly and has the following features.
  • Double Booking avoidance - if a booking was made on a location, date and time on a slot that has already been occupied the new booking will not be submitted. An alert will also pop-up to alert the user that the slot is already occupied.
  • Ability to Edit and Delete - if the user has admin privileges, the user can edit or delete a booking.
  • Past Date Selection Alarm - If the user has chosen a past date, an alert will be pop up to alert the user that the date chosen has already passed.
  • Sorting - the user can sort the bookings table by date, location and client.
